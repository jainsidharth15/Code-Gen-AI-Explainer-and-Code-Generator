# Code Gen AI Explainer and Code Generator with OCR

This project is an AI-powered educational tool that explains and generates source code.
It supports both text-based and image-based inputs using OCR technology.

## Features
- Code generation from user prompts
- Beginner-friendly code explanations
- OCR support for extracting code from images
- Automatic correction of syntax errors
- Streamlit-based web interface

## Technologies Used
- Python
- Streamlit
- Ollama (LLM)
- Tesseract OCR
- pytesseract
- OpenCV
